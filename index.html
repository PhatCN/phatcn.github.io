Bootstrap là gì?

Bootstrap là một framework giúp chúng ta xây dựng các website responsive một cách nhanh chóng, thuận tiện cũng như tiết kiệm nhiều công sức viết mã hơn

Bootstrap bao gồm các mẫu thiết kế dựa trên HTML và CSS cho các kiểu chữ, biểu mẫu, nút, bảng, điều hướng, hình ảnh..., cũng như các plugin JavaScript tùy chọn.
